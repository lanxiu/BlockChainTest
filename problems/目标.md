## 自建 Faucet 服务的基本原理：
无论你用哪种链，都可以写一个 Web 服务充当 faucet：

~~~
  javascript
  Copy
  Edit
  // 简单Node.js实现
  app.post("/faucet", async (req, res) => {
    const { address } = req.body;
    const tx = await web3.eth.sendTransaction({
      from: FAUCET_ADDRESS,
      to: address,
      value: web3.utils.toWei("0.1", "ether"),
    });
    res.send({ tx });
  });
